Tutorial: ACC

Learning Objectives

  1. How to create a table view with hierarchically decomposed rows (cells), where hierarchy is represented by indentation.
  2. How to show a down arrow in a table view row to indicate that the row has child rows. Tapping such a row expands (i.e., shows its child rows within the table view). Tapping an expanded row shrinks it (i.e., hides its child rows within the table view).
  3. How to make the table view row name (text label) to wrap to the next line in the table view.
  4. How to designate a leaf row (cell) as selected by changing its text label (name) color. (Leaf row is the one with no further decomposition.)
  5. How to show an icon in each table view row.
  6. How to read in a Property List XML file (plist file) from the main bundle (project folder), store its contents into dictionaries and arrays, and use them for your application logic processing.
  7. How to read in a Property List XML file (plist file) from a networked server computer, store its contents into dictionaries and arrays, and use them for your application logic processing.

Step 1: Application Functionality Specification

Upon launch, the app displays a list of the 14 ACC universities with their logos as shown in the following screen shot. The down arrow indicates that the row for a university is expandable. Tapping a university name reveals two more rows for sport categories, “Men's Sports” and “Women's Sports”, with their icons. The right arrow (disclosure indicator) implies that upon selection of a sport name, the website for that sport will be shown in another view. Tapping an expanded row shrinks it. Step 6: Target-Action Design Pattern

Table View Connection

Under the Target-Action design pattern, we store the object reference (unique ID) of the UITableView object into an instance variable in our code by way of graphical connection.

  1. Open the Main.storyboard file and the TableViewController.swift file in the Assistant Editor side by side.
  2. Click the Table View object in the Document Outline to select it.
  3. Control-drag from the Table View object to the top of the TableViewController.swift file where you want to insert the code. When you let go the mouse button, a dialog box shows up.
  4. Select Outlet from the Connection pop-up menu.
  5. Enter accSportsTableView as the Name, select UITableView as the Type, and click Connect. This connection inserts the following code:

    @IBOutlet var accSportsTableView: UITableView!

With this connection, we obtain the object reference (unique ID) of the Table View object in our code so that we can access its properties and invoke its methods.

Web View Connection

Under the Target-Action design pattern, we store the object reference (unique ID) of the UIWebView object into an instance variable in our code by way of graphical connection.

  1. Open the Main.storyboard file and the WebsiteViewController.swift file in the Assistant Editor side by side.
  2. Click the Web View object in the Document Outline to select it.
  3. Control-drag from the Web View object to the top of the WebsiteViewController.swift file where you want to insert the code. When you let go the mouse button, a dialog box shows up.
  4. Select Outlet from the Connection pop-up menu.
  5. Enter webView as the Name, select UIWebView as the Type, and click Connect. This connection inserts the following code:

    @IBOutlet var webView: UIWebView!

With this connection, we obtain the object reference (unique ID) of the Web View object in our code so that we can send a message to it and ask it to load a web page for a given URL.

UIWebView has a protocol called UIWebViewDelegate Protocol naming some methods that can be implemented optionally. You must implement the following three protocol methods whenever you display a UIWebView object.

func webViewDidStartLoad(webView: UIWebView!)

func webViewDidFinishLoad(webView: UIWebView!)

func webView(webView: UIWebView!, didFailLoadWithError error: NSError!)

To be able to capture and implement these protocol methods, we need to:

  1. Make the WebsiteViewController class to conform to the UIWebViewDelegate protocol by adding UIWebViewDelegate to the class declaration as shown below:

    class WebsiteViewController: UIViewController, UIWebViewDelegate {

  2. Designate the WebsiteViewController object to be the delegate for the UIWebViewDelegate protocol. To do this,

    • Control-drag from the UIWebView object to the WebsiteViewController object in the scene dock.
    • When you let go the mouse button, a dialog box shows up.
    • Select delegate from the pop-up menu to set the WebsiteViewController object to be the delegate for the UIWebViewDelegate protocol.

import UIKit

 

class TableViewController: UITableViewController {

   

    /*

    The accSports.plist XML file contains 3 dictionaries embedded into each other:

   

    Dictionary 1: Key = University Name    Value = object reference of Dictionary 2

    Dictionary 2: Key = Sport Category     Value = object reference of Dictionary 3

    Dictionary 3: Key = Sport Name         Value = sport's website URL

   

    The accUniversityLogos.plist XML file contains one dictionary:

   

    Dictionary 4: Key = University Name    Value = University Logo File Name

    */

   

    //---------- Create and Initialize the Dictionaries -----------------

   

    var dict1_UniversityName_Dict2  = [String: AnyObject]()

    var dict2_SportCategory_Dict3   = [String: AnyObject]()

    var dict3_SportName_SportURL    = [String: AnyObject]()

   

    var dict4_UniversityName_UniversityLogoFileName  = [String: String]()

   

    //---------- Create and Initialize the Arrays -----------------------

   

    var universityNames     = [String]()

    var sportCategoryNames  = [String]()

    var sportNames          = [String]()

   

   

    //Obtain the object reference to the UITableView object

    @IBOutlet var accSportsTableView: UITableView!

   

    /*

    When a table view row is expanded, new rows are inserted after it into the table view.

    When a table view row is shrunk, the rows expanded under it are deleted.

   

    These insertions and deletions will change table view list. We define

    tableViewList as a changeable array to hold the current rows of the table view.

    */

    var tableViewList = [String]()

   

    // The URL of the sport website to be passed to the WebsiteViewController object

    var urlOfWebsite: String = ""

   

    // Index Paths of the selected table view rows

    var selectedIndexPath: NSIndexPath = NSIndexPath()

    var selectedIndexPathPrevious: NSIndexPath = NSIndexPath()

   

    // Flags created and initialized

    var sportNameSelected: Bool = false

    var selectedRowNumber: Int = 0

   

    /*

    --------------------------

    MARK: - viewDidLoad Method

    --------------------------

    */

    override func viewDidLoad() {

       

        super.viewDidLoad()

       

        //-------------- accSports -----------------

       

        // Obtain the file path to the plist file. mainBundle refers to your project module.

        var accSportsFilePath: NSString? = NSBundle.mainBundle().pathForResource("accSports", ofType: "plist")

       

        // NSDictionary manages an *unordered* collection of key-value pairs. Instantiate an

        // NSDictionary object and initialize it with the contents of the accSports.plist file.

        var dictionary1FromFile: NSDictionary? = NSDictionary(contentsOfFile: accSportsFilePath!)

       

        if let dictionaryForAccSportsPlistFile = dictionary1FromFile {

           

            // Typecast the created dictionary of type NSDictionary as Dictionary type

            // and assign it to the class property named dict1_UniversityName_Dict2

            dict1_UniversityName_Dict2 = dictionaryForAccSportsPlistFile as Dictionary

           

            /*

            allKeys returns a new array containing the dictionary’s keys as of type AnyObject.

            Therefore, typecast the AnyObject type keys to be of type String.

            The keys in the array are *unordered*; therefore, they need to be sorted.

            */

            universityNames = dictionaryForAccSportsPlistFile.allKeys as [String]

           

            // Sort the universityNames within itself in alphabetical order

            universityNames.sort { $0 < $1 }

            /*

            // Print the sorted university names

            println(universityNames)

           

            // Print the entire dictionary: dict1_UniversityName_Dict2

            for (key, value) in dict1_UniversityName_Dict2 {

            println("\(key): \(value)")

            }

            // You can also print a dictionary using: println(dict1_UniversityName_Dict2)

            */

           

        } else {

           

            // Unable to get the file from the main bundle

            showErrorMessageFor("accSports.plist")

            return

        }

       

        //-------------- accUniversityLogos -----------------

       

        // Obtain the file path to the plist file. mainBundle refers to your project module.

        var accUniversityLogosFilePath: NSString? = NSBundle.mainBundle().pathForResource("accUniversityLogos", ofType: "plist")

       

        // Instantiate an NSDictionary object and initialize it with the contents of the accUniversityLogos.plist file.

        var dictionary4FromFile: NSDictionary? = NSDictionary(contentsOfFile: accUniversityLogosFilePath!)

       

        if let dictionaryForAccUniversityLogosPlistFile = dictionary4FromFile {

           

            // Typecast the created dictionary of type NSDictionary as Dictionary type and assign it.

            dict4_UniversityName_UniversityLogoFileName = dictionaryForAccUniversityLogosPlistFile as Dictionary

           

        } else {

           

            // Unable to get the file from the main bundle

            showErrorMessageFor("accUniversityLogos.plist")

            return

        }

       

        //-------------- sportNames --------------

       

        // Obtain the file path to the plist file. mainBundle refers to your project module.

        var sportNamesFilePath: NSString? = NSBundle.mainBundle().pathForResource("sportNames", ofType: "plist")

       

        // Instantiate an NSArray object and initialize it with the contents of the sportNames.plist file.

        var arrayFromFile: NSArray? = NSArray(contentsOfFile: sportNamesFilePath!)

       

        if let arrayForSportNamesPlistFile = arrayFromFile {

           

            // Typecast the created array of type NSArray as Array type and assign it.

            sportNames = arrayFromFile as Array

            /*

            // Print the sorted university names

            println(sportNames)

            */

           

        } else {

           

            // Unable to get the file from the main bundle

            showErrorMessageFor("sportNames.plist")

            return

        }

       

       

        //-------------- Initialize the table view list --------------

       

        // Initialize the current table view rows to be the list of the university names

        tableViewList = universityNames

    }

   

    /*

    ------------------------------------------------

    MARK: - Show Alert View Displaying Error Message

    ------------------------------------------------

    */

        // Instantiate an alert view object

        var alertView = UIAlertView()

       

        alertView.title = "Unable to Access the File: \(fileName)!"

        alertView.message = "The file does not reside in the application's main bundle (project folder)"

        alertView.delegate = nil

        alertView.addButtonWithTitle("OK")

       

        alertView.show()

    }

   

    /*

    ---------------------------

    MARK: - View Appear Methods

    ---------------------------

    */

   

    // Prepare the view before it appears to the user

    override func viewWillAppear(animated: Bool) {

       

        if sportNameSelected {

           

            // Reload the table view data so that the selected sport name can be

            // colored in blue to indicate that it is the selected row.

            accSportsTableView.reloadData()

        }

       

        super.viewWillAppear(animated)

    }

   

    // Scroll the selected sport name row towards the middle of the table view

    override func viewDidAppear(animated: Bool) {

       

        if sportNameSelected {

           

            // Scroll the selected row towards the middle of the table view

            accSportsTableView.scrollToRowAtIndexPath(selectedIndexPath,

                atScrollPosition: UITableViewScrollPosition.Middle, animated: true)

        }

       

        super.viewDidAppear(animated)

    }

   

    /*

    --------------------------------------

    MARK: - Table View Data Source Methods

    --------------------------------------

    */

   

    /*

    A Table View object (UITableView) consists of Sections.

    A Section consists of Rows.

    A Row contains one Cell object (UITableViewCell).

    A Cell object is configured with the UI objects (e.g., image, text, accessory icon) to create the look of a row.

    */

   

    // Asks the data source to return the number of sections in the table view

    override func numberOfSectionsInTableView(tableView: UITableView) -> Int {

       

        return 1

    }

   

    // Asks the data source to return the number of rows in a section, the number of which is given

    override func tableView(tableView: UITableView, numberOfRowsInSection section: Int) -> Int {

       

        return tableViewList.count

    }

   

    /*

    ------------------------------------

    Prepare and Return a Table View Cell

    ------------------------------------

    */

    override func tableView(tableView: UITableView, cellForRowAtIndexPath indexPath: NSIndexPath) -> UITableViewCell {

       

        var rowNumber: Int = indexPath.row    // Identify the row number

       

        // Obtain the object reference of a reusable table view cell object instantiated under the identifier

        // TableViewCellReuseID, which was specified in the storyboard

        var cell: UITableViewCell = tableView.dequeueReusableCellWithIdentifier("TableViewCellReuseID") as UITableViewCell

       

        // Obtain the name of the row from the table view list

        var rowName: String = tableViewList[rowNumber]

       

        // Set the label text of the cell to be the row name

        cell.textLabel!.text = rowName

       

        // Cell indentation width of 10.0 points is the default value

        cell.indentationWidth = 10.0

       

        if contains(universityNames, rowName) {    // Rows decomposition level = 0

           

            // Row name is a university name

            cell.indentationLevel = 0;

           

            var universityLogoFileName: String? = dict4_UniversityName_UniversityLogoFileName[rowName]

           

            cell.imageView!.image = UIImage(named: universityLogoFileName!)

           

        } else if rowName == "Men's Sports" {           // Rows decomposition level = 1

           

            cell.indentationLevel = 1

            cell.imageView!.image = UIImage(named:"MenSportIcon")

           

        } else if rowName == "Women's Sports" {         // Rows decomposition level = 1

           

            cell.indentationLevel = 1

            cell.imageView!.image = UIImage(named:"WomenSportIcon")

           

        } else {                                        // Rows decomposition level = 2

           

            // Row name is a sport name

            cell.indentationLevel = 2

           

            // Image name is the same as the sport name

            cell.imageView!.image = UIImage(named: rowName)

        }

       

        return cell

    }

   

    /*

    --------------------------------------------------------------

    Prepare the Table View Cell before it is displayed to the user

    --------------------------------------------------------------

    */

    override func tableView(tableView: UITableView, willDisplayCell cell: UITableViewCell, forRowAtIndexPath indexPath: NSIndexPath) {

       

        // Set the number of lines to be displayed in each table view cell to 2

        cell.textLabel!.numberOfLines = 2

       

        // Set the text to wrap around on the next line

        cell.textLabel!.lineBreakMode = NSLineBreakMode.ByWordWrapping

       

        // Set the cell label text to use the System font of size 14 pts

        cell.textLabel!.font = UIFont(name: "System", size: 14.0)

       

        if sportNameSelected && (selectedRowNumber == indexPath.row) {

           

            // Set the cell label text color to blue to indicate its selection

            cell.textLabel!.textColor = UIColor.blueColor()

           

        } else {

            // Set the cell label text color to black otherwise

            cell.textLabel!.textColor = UIColor.blackColor()

        }

       

        switch cell.indentationLevel {

        case 0:

            // Set level 1 (University Name) row background color to Lavendar (#E6E6FA)

            cell.backgroundColor = UIColor(red: 230.0/255.0, green: 230.0/255.0, blue: 250.0/255.0, alpha: 1.0)

           

        case 1:

            // Set level 2 (Sport Category Name) row background color to Ivory (#FFFFF0)

            cell.backgroundColor = UIColor(red: 255.0/255.0, green: 255.0/255.0, blue: 240.0/255.0, alpha: 1.0)

           

        case 2:

            // Set level 3 (Sport Name) row background color to PeachPuff (#FFDAB9)

            cell.backgroundColor = UIColor(red: 255.0/255.0, green: 218.0/255.0, blue: 185.0/255.0, alpha: 1.0)

           

        default:

            println("Cell indentation level is out of range!")

            break

        }

       

        // If the cell label is a sport name

        if contains(sportNames, cell.textLabel!.text!) {

           

            // Then, show the Right Arrow image as Disclosure Indicator

            cell.accessoryView = UIImageView(image: UIImage(named: "RightArrow"))

           

        } else {

           

            // Otherwise, show the Down Arrow image to indicate that the row has child rows

            cell.accessoryView = UIImageView(image: UIImage(named: "DownArrow"))

        }

    }

   

    /*

    ----------------------------------

    MARK: - Table View Delegate Method

    ----------------------------------

    */

   

    /*

    ---------------------------------------------------------------------------

    Perform the necessary actions when the user selects a table view row (cell)

    ---------------------------------------------------------------------------

    */

    override func tableView(tableView: UITableView, didSelectRowAtIndexPath indexPath: NSIndexPath) {

       

        // Obtain the number of the selected row

        var rowNumber: Int = indexPath.row

       

        // Set the class properties

        selectedIndexPath = indexPath

        selectedRowNumber = rowNumber

       

        // Obtain the name of the selected row

        var nameOfSelectedRow: String = tableViewList[rowNumber]

       

        // Identify the table view cell object for the selected row

        var selectedCell: UITableViewCell = tableView.cellForRowAtIndexPath(indexPath)!

       

        // Obtain the indentation level of the selected row

        var indentationLevelOfSelectedCell: Int = selectedCell.indentationLevel

       

        switch indentationLevelOfSelectedCell {

           

            /*

            -------------------------------------------------------------------------

            University Name is Selected: Expand the row to show its Sport Categories.

            -------------------------------------------------------------------------

            */

        case 0:

           

            sportNameSelected = false

           

            // If the selected university name is the last row or

            // If the row below the selected university name is also a university name, implying that it is not expanded

            if rowNumber == (tableViewList.count) - 1 || contains(universityNames, tableViewList[rowNumber+1]) {

               

                // Expand the selected university name row

import UIKit

 

class WebsiteViewController: UIViewController, UIWebViewDelegate {

   

    // Object reference to the Web View object

    @IBOutlet var webView: UIWebView!

 

    // The URL of the web page to be displayed. Initial value is empty.

    // It will be set by the previous view controller just before this view is shown

    var websiteURL: String = ""

   

    override func viewDidLoad() {

       

        super.viewDidLoad()

       

        /*

        Convert the websiteURL set by the previous view controller into an NSURL object and

        store its object reference into the local variable url. An NSURL object represents a URL.

        */

        var url = NSURL(string: websiteURL)

       

        /*

        Convert the NSURL object into an NSURLRequest object and store its object

        reference into the local variable request. An NSURLRequest object represents

        a URL load request in a manner independent of protocol and URL scheme.

        */

        var request = NSURLRequest(URL: url)

       

        // Ask the webView object to display the web page for the given websiteURL

        webView.loadRequest(request)

    }

   

    /******************************************************************************************

    * UIWebView Delegate Methods: These methods must be implemented whenever UIWebView is used.

    ******************************************************************************************/

   

    func webViewDidStartLoad(webView: UIWebView!) {

        // Starting to load the web page. Show the animated activity indicator in the status bar

        // to indicate to the user that the UIWebVIew object is busy loading the web page.

        UIApplication.sharedApplication().networkActivityIndicatorVisible = true

    }

   

    func webViewDidFinishLoad(webView: UIWebView!) {

        // Finished loading the web page. Hide the activity indicator in the status bar.

        UIApplication.sharedApplication().networkActivityIndicatorVisible = false

    }

   

    func webView(webView: UIWebView!, didFailLoadWithError error: NSError!) {

        /*

        Ignore this error if the page is instantly redirected via javascript or in another way.

        NSURLErrorCancelled is returned when an asynchronous load is cancelled, which happens

        when the page is instantly redirected via javascript or in another way.

        */

        if error.code == NSURLErrorCancelled {

            return

        }

       

        // An error occurred during the web page load. Hide the activity indicator in the status bar.

        UIApplication.sharedApplication().networkActivityIndicatorVisible = false

       

        // Create the error message in HTML as a character string and store it into the local constant errorString

        let errorString = "<html><font size=+2 color='red'><p>An error occurred: <br />Possible causes for this error:<br />- No network connection<br />- Wrong URL entered<br />- Server computer is down</p></font></html>" + error.localizedDescription

       

        // Display the error message within the UIWebView object

        // self. is required here since this method has a parameter with the same name.

        self.webView.loadHTMLString(errorString, baseURL: nil)

    }

   

}

Step 10: Access the Input Data Files from a Server Computer over the Network

Reading input data from the app's main bundle (project folder) creates maintainability problems. When the input data changes, the app is required to have an update. The best practice calls for decoupling the data from the app. We can implement the best practice by placing the input data files on a server computer and reading them into our app by accessing them over the network. This way, we can change the data anytime necessary without affecting the functionality of our app or requiring the user to download an updated version of our app. To accomplish this, do the following:

  1. Copy your ACC project folder and paste it under the name ACC Net
  2. Open your ACC Net app under Xcode
  3. Click the Plist Files group (containing accSports.plist, accUniversityLogos.plist, and sportNames.plist) in the Project Navigator to select it, press the Delete key, and select Move to Trash from the dialog box to delete them all from our project.
  4. In the TableViewController.swift file, replace the viewDidLoad method and the showErrorMessageFor method with the following code given below.
